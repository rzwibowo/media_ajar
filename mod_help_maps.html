Kembali ke Menu Utama

Untuk kembali ke menu utama, klik tombol Kembali di kiri atas.

Menutup Menu Bantuan

Untuk menutup kotak bantuan ini, klik tombol X dengan kotak merah.

Penggunaan Peta

  1. Arahkan kursor ke area Peta
  2. Klik provinsi tersebut, akan tampil informasi mengenai provinsi tersebut
  3. Untuk menutup informasi provinsi, klik tombol X dengan lingkaran merah

Perbesaran Peta

Untuk memperjelas tampilan peta, kamu bisa menggunakan fitur perbesaran, caranya:

  1. Klik tombol kaca pembesar dengan simbol +
  2. Perbesaran aktif, ditandai dengan berubahnya simbol kaca pembesar dari + menjadi –. Arahkan kursor ke peta, tunjuk ke salah satu provinsi
  3. Klik provinsi tersebut, akan tampil informasi mengenai provinsi tersebut
  4. Untuk menutup informasi provinsi, klik tombol X dengan lingkaran merah
  5. Untuk menonaktifkan fitur perbesaran, klik tombol kaca pembesar dengan simbol –

Memilih Bentuk Peta Lain

Dalam menu Peta Indonesia, disediakan bentuk peta lain yaitu Peta Batas Wilayah dan Peta Letak Indonesia. Untuk menampilkan peta lain langkahnya adalah sebagai berikut:

  1. Klik pada tombol Lihat Peta Lain
  2. Akan muncul pilihan peta, silakan klik salah satu dari pilihan peta tersebut
  3. Untuk kembali ke tampilan awal peta, klik tombol Peta Indonesia (Default)
  4. Untuk menutup pilihan peta, klik tombol Lihat Peta Lain
